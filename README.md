# Bank Loan Approval Performance Dashboard

## Project Overview
This project analyzes bank loan approval patterns using Python and Power BI.

The objective is to identify key factors influencing loan approvals and rejection trends.

---

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- DAX
- Data Cleaning & Feature Engineering

---

## Key KPIs
- Approval Rate
- Rejection Rate
- Average Loan Amount
- Average Income Loan Ratio
- Total Applications
- Average Total Income

---

## Insights
- Applicants with Credit History = 1 have significantly higher approval rates.
- Semiurban areas show highest application volume.
- Education level impacts approval probability.
- Urban and Rural areas show moderate approval differences.
